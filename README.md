 Final Project Visi Komputer  
Image Matching Menggunakan SIFT (Scale-Invariant Feature Transform)

 Deskripsi
Repository ini berisi implementasi algoritma **Scale-Invariant Feature Transform (SIFT) untuk melakukan image matching antara dua citra.  
Project ini bertujuan untuk mendeteksi keypoints, mengekstraksi descriptor, dan mencocokkan fitur secara robust terhadap perubahan skala, rotasi, dan pencahayaan.

Project dikembangkan menggunakan bahasa Python dan library OpenCV, sebagai bagian dari tugas uas Mata Kuliah Visi Komputer.
